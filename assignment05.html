\[5000\times0 + 10000\times0.10 + 20000\times0.15 + 3000\times0.2 = 4600 \mbox{tvarps.}\]

Q2.

An area of unit circle can be approximated using Monte Carlo methods. Monte Carlo methods are one of the most important areas of scientific computing. Along the way we will also learn how to generate random numbers (technically pseudo-random numbers) in C++.

Essentially idea is as follows. Let us create a lot of random points in a square \([0,1]\times[0, 1]\). A subset of these points will lie in a cricle centered at \(0.5\) with a radius of \(0.5\). We can count the number of points that fall inside the circle. Then as the total number of points increases, area of the circle can be approximated by

\[ \frac{\mbox{# of points inside the circle}}{\mbox{Total # of points}} = \frac{\mbox{Area of circle}}{\mbox{Area of unity square}}\].

Since we know that the radius of circle is \(0.5\), this method can also be viewed as a method of approximating the value of \(\pi\).

There are a few key things to notice.

  1. Random numbers can be generated in C++ using rand() defined in cstdlib (Câ€™s stdlib library). Since computers are deterministic in nature, true random numbers cannot be generated on computers. We generate what are called as Pseudorandom numbers. Many specialised algorithms exists for this1. rand() generates uniformly distributed integer random number in the range of \(0\) to RAND_MAX (which is defined in cstdlib header file). Following code is a standard way of generating random number of type double in the range \([0, 1]\).
#include <cstdlib>

double randomNumber;
randomNumber = double(rand())/RAND_MAX;
  1. rand() generates a sequence of integers randomly. However, it is not truly random. It depends on the seed provided to rand(). Ordinarily, rand() function will use the current system clock time as seed. Hence, every time we call the function, we get a different sequence. However, seed can be set to generate the same sequence of random numbers(ðŸ˜„). The seed is set using srand at the beginning of the program.
srand(1);
rand();
  1. Generating a random point in the square \([0,1]\times[0, 1]\) means picking the x and y coordinates of the point randomly. Hence, two calls to the rand() function will be required to generate a point.

The input to the program is the precision to which the area is to be calculated and the output of the program should be the value of the area of circle. Output 15 digit precise solution using setprecision(15). Note the difference in the meaning of the word precision. You have to understand the meaning from the context.
