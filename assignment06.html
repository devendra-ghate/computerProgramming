This week we will be working with text and not numbers. We define a frequency of a word (or a character) as the total number of times that word occurs in a text file. Write a program using <fstream> to read a file!sidenote(Name of the input text file is to be fixed to sample.txt. A sample text file can be found here.) and report back:

  1. Total number of characters (noc)!sidenote(A character here means any english alphabet. It excludes punctuation marks.)
  2. Total number of words (now)
  3. Total number of lines (nol)
  4. Frequency of all vowels (fv[5])
  5. Frequency of a word (input by user) (nofw)

Input to the program will be the word whose frequency is to be calculated. A sample output is:

Guidelines

  1. Your code should use following function prototypes.
#include<iostream>
#include<fstream>
using namespace std;
// to find the word and returns numbers of its occurrences.
int find_word(ifstream &fin, char search[]);

// to find the number of characters and return its total.
int no_of_characters(ifstream &fin);

// to find the number of words and return its total.
int no_of_words(ifstream &fin);

// to find the number of lines and return its total.
int no_of_lines(ifstream &fin);

// to find frequency of each vowels and stores to array fv[].
void frequency_of_vowels(ifstream &fin, int fv[]);

void display(int &nofw, int &noc, int &now, int &nol, int fv[]);
  1. You have to call all these functions from main().
  2. Do not use any header files other than <iostream> and <fstream>.
  3. Opening and closing of file should be inside main() function.

A few hints:

  1. A file can be read in three ways: character by character, word by word, and line by line.
//  read file character by character
while(!fin.eof())
{ char c=fin.get(); }

//  read each word in a file
// (space, tab, newline act as a delimiter).
char word[30];
while(fin>>word) { ... }

// read each line ends with newline character
char line[1000];
while(fin.getline(line,1000)) { ... }
  1. When the file pointer reaches the end of the file, a flag 'eof' is set and it always returns false. In such conditions, we have to clear the flag and move file pointer to beginning of the file.
// clear eof flag
fin.clear();
// file seek to beginning of the file.
fin.seekg(0,ios::beg);
